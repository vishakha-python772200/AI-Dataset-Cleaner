# AI Dataset Cleaner

This project automatically cleans datasets and generates visualizations.

## Features
- Data Cleaning
- Missing Value Handling
- Outlier Detection
- Data Visualization
- Dataset Type Prediction

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Run Project

Install libraries

pip install -r requirements.txt

Run project

python app.py
